Multi Branch Accounting Reports

This application provide functionality of filter enterprise accounting reports by branches.

This module is compatible with

  • Community
  • Enterprise
  • Odoo.sh

Key Features

    Branch Filter on Profit and Loss Report.
    Branch Filter on Balance Sheet Report.
    Branch Filter on Cash Flow Statement.
    Branch Filter on Executive Summary.
    Branch Filter on Partner Ledger.
    Branch Filter on General Ledger.
    Branch Filter on Aged Receivable.
    Branch Filter on Aged Payable.
    Branch Filter on Trial Balance.
    Generate report branch wise.

User can see Single Branches, Multiple Branches Report
