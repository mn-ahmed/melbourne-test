Multi Branches For Point Of Sale

This application provides functionality to manage multi branches for a single company.
Multi branches functionality for Point of Sale.
Also maintains user and manager level access rights.

Key Features

  • Define multiple branches for companies
  • Multiple branches working same as Odoo multi company scenario
  • Multi branches on Point of Sale - users sessions
  • Multi branches on Point of Sale - Stock Picking
  • Multi Branches On Users' opening balance
  • Branches wise Point of sale receipts generations
  • Branches wise Point of sale invoices generations

Configuration

Creation of branch will automatically create warehouse, locations and partner related to branch.

Assigning a branch to user

You can assign multiple branches to specific users. Current branch is filtered on related company and allocated branches.
For multi branches access, need to allocate 'Multi Branches' group in Users. Filter all data as per current branches.

Branches On Point Of Sale Configurations

Branches are defined on users related Point of Sales configurations.
Invoice Journals & Inventory Locations will be set related to the branch.

Branch on Point of Sale opening and closing balance

Each user's opening as well as closing balance can be easily bifurcated via their related Branch

Branch On Point Of Sale Orders

When a Sales order will be generated from Point of Sale screen then it will be bifurcated automatically related to user's branch.
Point of Sale orders will be grouped by different branches.

Branch On Point Of Sale Stock Pickings

When a stock picking is generated from Point of Sale order then it will be bifurcated automatically related to user's Branch.

Branch On Invoices

When an Invoice is generated from Point of Sale order then it will be bifurcated automatically related to user's branch

Branch wise Point of Sale sessions summary

Point of Sale manager will be able to see different user's sessions by their related branches.

Branch wise Point of Sale Receipts

Point of sale receipts will be bifurcated branch wise.

Branch wise Point of Sale invoices summary

Point of sale invoices will be bifurcated different branch wise.
